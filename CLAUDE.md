# HM 2026 — Dystir & Sjónvarp

A single-page web app for Faroese football fans following the 2026 FIFA World Cup. Shows match schedules for selected teams in **Faroese time**, with the TV channels available in the Faroe Islands.

## What it is

- Single HTML file: `index.html` (all CSS and JS inline)
- Language: Faroese (`lang="fo"`)
- No build tools, no frameworks — pure HTML/CSS/JS
- Hosted on GitHub: https://github.com/steingmo/hm2026

## Teams covered

Germany, England, Netherlands, Brazil, Argentina, Spain, France, Norway, Sweden, Scotland, Portugal

## Broadcasters shown

- **KVF** — Kringvarp Føroya (Faroese national broadcaster)
- **DR** — Danmarks Radio
- **TV 2 Play** — Danish streaming

## Key features

- Match schedule per team with Faroese kickoff times
- TV broadcast badges per match
- Sticky right sidebar listing all matches chronologically
- Interactive calendar sidebar — click a day to see matches
- Subtitle team list (teams with Faroese commentary/subtitles)
- Dark theme with yellow/orange accent colors

## Fonts

- **Bricolage Grotesque** — body text
- **Fraunces** — display/headings
- **JetBrains Mono** — labels, eyebrow text

## Development notes

- Match data is hardcoded in the HTML — no external data source
- Calendar and sidebar are generated by inline JavaScript at bottom of file
- `hm2026.html` is an identical mirror of `index.html` — **every change to `index.html` must also be applied to `hm2026.html`** (they are kept byte-identical; `cp index.html hm2026.html` after verifying they were in sync)
- All times are in Faroese local time (UTC+0 in winter, UTC+1 in summer)
